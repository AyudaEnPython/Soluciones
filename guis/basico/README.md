# Enunciado Original

1. Armar una aplicación que permita ingresar un nombre en un cuadro de texto
  y luego de apretar un botón, muestre en una etiqueta "Hola" + el nombre
  ingresado.

2. Armar una aplicación que permita sumar 2 números y al oprimir un botón,
  calcule y muestre el resultado.

3. Armar una aplicación que me permita ingresar el nombre, el precio y la
  cantidad de un producto que se va a comprar y, luego de apretar un botón,
  muestre en una etiqueta cuál es el precio que debe pagar por la compra.

4. Armar una aplicación que permita ingresar 3 numeros distintos en 3 cuadros
  de texto y, luego de apretar un botón, me muestre en 3 etiquetas los números
  ordenados de mayor a menor.

5. Armar una aplicación que simule el ingreso a la plataforma de "AulaPerez",
  que pida usuario y contraseña y señale si la contraseña es correcta o no luego
  de apretar un botón de ingreso.
